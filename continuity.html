Definition

Our eyes and mind will follow lines and continue patterns past where they end (Hagen and Golombisky 62).

In Summary

If we see a pattern then we tend to continue that pattern, because we just assume that the pattern is continuing. This is how you create shapes and designs using different shapes and elements without them looking disjointed.

Real World Context

Use this theory to better guide a consumers eyes across a layout. This can be used well in navigation, by consistently using shapes to guide what the user sees.
